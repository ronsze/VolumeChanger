<H1>소개</H1>
VolumeChanger는 사용자의 위치에 따라서 자동적으로 스마트폰 기기의 볼륨을 조절해주는 App입니다.  

사용자는 자신이 원하는 위치와 볼륨을 설정하고, App은 Geofence를 통해 사용자 위치를 감시하여 볼륨을 조절합니다.
